# TradingSonar
**A modern alternative to TradingView for charting and market analysis**  
Built with traders, for traders — unlimited indicators, multi-layout system, and community-driven development.

---

## What is TradingSonar?
TradingSonar is a charting and analysis platform designed to solve the limitations traders face in existing tools.  
Our goal is to provide a professional environment **without paywalls on basic features**.

---

## Key Features

### Unlimited Indicators (Free)
- No indicator limits for any user.
- Fully free — no locked tools behind subscriptions.
- Add as many indicators as your analysis requires.

### Multi-Layout System (Free)
- Use multiple charts on a single screen.
- Save and restore personalized layouts.
- Completely free for all users.

### Sonar Analysis (Plus Membership)
Advanced tools focused on actionable insights:
- Market signal scanning
- Price behavior analytics
- Automatic pattern/formation detection *(in development)*

Plus membership only applies to Sonar Analysis and advanced tools.  
Charts, indicators, and layouts are always free.
